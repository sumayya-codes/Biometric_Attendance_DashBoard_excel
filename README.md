# Biometric_Attendance_DashBoard_excel
An interactive Biometric Attendance Management Dashboard built in Microsoft Excel using Pivot Tables, Pivot Charts, Slicers, and Excel formulas to analyze employee attendance, overtime, late arrivals, department-wise performance, and workforce insights.
